The goal of this project is to bridge the gap between dense accounting data and executive decision-making. By converting a 3-statement financial model into visual components, this project highlights key trends, identifies anomalies (like the 2016 income spike), and illustrates the 5-year forecast trajectory. It turns "rows and columns" into a story of corporate growth and capital management.
